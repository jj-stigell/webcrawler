Web Scraping Used

Suurien tietomäärien keruuseen verkkosivuilta.
Strukturoidaan kerätty tieto esim csv tiedostoon.

Verkkosivu voi määritellä robot.txt tiedostossaan onko keräily sallittua.
verkkosivu.com/robots.txt

To extract data using web scraping with python, you need to follow these basic steps:

1. Find the URL that you want to scrape
2. Inspecting the Page
3. Find the data you want to extract
4. Write the code
5. Run the code and extract the data
6. Store the data in the required format 

Hommaan käytetyt kirjastot:

Selenium https://www.selenium.dev/
- Webtesting library. selain toimintojen automatisointiin.

BeautifulSoup https://www.crummy.com/software/BeautifulSoup/bs4/doc/
- helpottaa tiedon keräämistä html/xml tiedostoista.

Pandas https://pandas.pydata.org/
- python kirjasto datan manipulointiin ja analysointiin. voidaan kerätä ja tallentaa
dataa haluttuun muotoon.
